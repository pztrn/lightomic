# A 3(2)-column Theme for StatusNet

## Screenshot

![Screenshot](https://github.com/pztrn/lightomic/raw/master/screenshot.png)

## Instructions

Drop the files in your `/theme/` folder  
Tell `/config.php` to use it with: `$config['site']['theme'] = 'lightomic';`
